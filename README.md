## Development Environment 
- Clone this [repository](https://github.com/gh0stfrk/TextMask.git)

```bash
git clone https://github.com/gh0stfrk/TextMask.git
```
- Install node dependencies 
```bash
npm install
```

- Run the development server
```
npm run dev
```
